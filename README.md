# Image Server
このサーバーコードは、`puk06/Image-Uploader`を使用した画像アップロードに使用できるサーバーを立てるためのコードです。

# 初回起動手順
1. `.env`ファイルを作成し、APIキーを作って入力する。
2. `npm i`でライブラリをインストール
3. `node ImageServer.js`で起動

# 通常時の起動手順
1. `node ImageServer.js`で起動するだけです。
